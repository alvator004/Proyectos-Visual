Lenguaje de Marcas, Modulo de 1º

Un lenguaje de marcas es un sistema de notación que utiliza etiquetas o marcas para estructurar y presentar información de manera que pueda ser leída tanto por humanos como por máquinas. Estas etiquetas definen la estructura, el formato y el significado de los elementos de un documento. En otras palabras, un lenguaje de marcas es una forma de codificar un documento incorporando etiquetas o marcas que contienen información adicional sobre la estructura o presentación del texto. Estas etiquetas ayudan a organizar el contenido, definir elementos específicos y proporcionar instrucciones para su presentación visual.

Programación, Modulo de 1º

La programación informática es el proceso de diseñar, escribir, probar y mantener el código de software para que una computadora pueda realizar una tarea específica. Es como dar instrucciones a una computadora en un lenguaje que ella puede entender, para que pueda hacer cosas útiles, como calcular, procesar datos, o controlar dispositivos.

Base de Datos, Modulo de 1º

Una base de datos informatica es una recopilación organizada de información almacenada electrónicamente, diseñada para ser consultada, actualizada y gestionada de forma eficiente. Permite almacenar, recuperar y editar datos de diversos tipos, como texto, números, imágenes, vídeos y archivos.

¿Donde puedes aprender a realizar páginas web?

Mi recomendación es llevar un estudio progresivo si es que de verdad te inteeresa el tema. Pero si tu único objetivo es diseñar una sola pagina web puedes copiar el codigo de un repositorio de github, o mirar tutoriales en youtube sobre la página que más te guste y editarla a tú manera.

¿Cuál es el objetivo de esta página?

Comenzo como un proyecto para el Grado, pero poco a poco se ha ido combirtiendo en mi proyecto personal para demostrarme de lo que soy capaz, y para mejorar en el diseño web y la programación.

¿Qué pasa al registrarse?

Obtienes acceso total al contenido de la página. Además de asegurar que los datos enlazados no serán compartidos, incluso puede dar datos falsos. El único objetivo del registro es comprobar la recogida de datos del sistema para su posterior manejo, mejorando y optimizando la página.
